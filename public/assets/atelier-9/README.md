# Atelier 9 images

Put the Atelier 9 / Boutique Spa Souissi images **here** so the landing page can use them.

Required filenames (from your `assets/atelier-9` folder):

- `atelier-9-reception.png`
- `atelier-9-reception-2.png`
- `atelier-9-skin-gym.png`
- `atelier-9-chairs.png`
- `atelier-9-chairs-2.png`
- `atelier-9-hall.png`
- `atelier-9-small-room.png`
- `atelier-9-wall-logo.png`

**If your images are in a different folder** (e.g. `assets/atelier-9` at the project root), **copy or move them into this folder**: `public/assets/atelier-9/`. Next.js only serves files that are inside `public/`.
